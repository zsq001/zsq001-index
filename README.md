# zsq001-index
This is source code of zsq001-index.  It's still on developing!

## Setup
### Install the dependencies
`yarn install`      

### Run 
`yarn serve`

### Build
`yarn build`, the `dist` directory is the root directory.          
When building the project for the first time, add parameter `--fix`.

## Technical stack
- Vue.js + mdui
